# djadigitalstudio-website

Static marketing site for djadigitalstudio.com — see `ARCHITECTURE.md`.

This repo also hosts `daily-affirmations/`, a separate, self-contained internal tool (own
`package.json`, own build) unrelated to the static site generator above — see
`daily-affirmations/README.md`. It is not part of the public website and the site's build
process (`build.js`) never touches it.
