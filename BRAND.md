# Brand reference — DJ&A Digital Studio

This is the canonical brand reference for djadigitalstudio.com. Where an
external asset sheet (e.g. one generated by another AI tool) disagrees with
this document, **this document wins** — it reflects what's actually live on
the website today, not an aspirational spec.

## Company

- **Legal entity:** DJ&A Digital Studio Limited
- **Trading name:** DJ&A Digital Studio
- **Country:** New Zealand
- **NZBN:** not yet assigned. Not displayed anywhere on the site (removed from Terms/EULA governing-law clauses and contact blocks). Do not add one until the Companies Office issues it — do not invent a number.

## Typography (official — do not change without an explicit decision)

- **Headings:** `Cormorant Garamond` (serif) — loaded via Google Fonts
- **Body / UI text:** `Josefin Sans` (sans-serif) — loaded via Google Fonts

An earlier AI-generated brand sheet listed Playfair Display + Poppins. That
was never adopted — the site has used Cormorant Garamond + Josefin Sans
since before this brand system existed, and that remains the standard.

### Header logo — responsive & retina

`.nav-logo-img`/`.footer-logo-img` are constrained by CSS (`height: 28px`/
`22px`; `width: auto`) rather than fixed HTML width/height attributes, so
they scale naturally at any viewport width — same treatment as the text
wordmark they replaced, no separate mobile breakpoint needed (nav padding
already adjusts at `max-width: 768px`; the logo itself doesn't need to).

Retina-ready: the source file behind both is 1536px wide at native
resolution. Displayed at a CSS height of 28px, the actual delivered image
is 120px tall (`logo-horizontal-web.png`) — about **4.3× the display
size**, well past the 2× a standard Retina/HiDPI screen needs, so it stays
sharp even on 3× displays. Same margin on the footer's smaller 22px display
height.

## Where the horizontal logo is NOT used (by design)

App-specific pages (`apps/{id}/`, `privacy/{id}/`, `terms/{id}/`) show the
**app's own name** in their footer (e.g. "ShiftEarn Pro"), not the DJ&A
logo — that's intentional per-app branding, unrelated to the company
wordmark question. Their nav bar still shows the DJ&A horizontal logo,
since nav is company-level navigation ("← All Apps") on every page.
